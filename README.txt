# Mission-reisbureau



Om te instaleren:

unzip de file.
importeer de mission-reisbureau.sql file in een php myadmin database als database.
vervolgens gebruik de hanzeproject map als om het project van te draaien en gebruik index.php als pagina om van te starten!!!
